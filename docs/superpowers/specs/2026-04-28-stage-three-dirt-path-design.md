# 第三阶段短土路设计

## 背景

在 `habit-planet` 的逐日成长场景中，用户希望在第三阶段补充一条低模、土色、轻微弯曲的小路，用来增强“星球顶部模型聚集区域开始出现生活痕迹”的叙事感。该小路不应成为主角，而应作为地表细节自然融入现有场景。

项目已有明确的“逐日累积”规则：后一天默认继承前一天已经出现的模型与表现，不主动删除已有内容。因此这条小路需要按总天数逐步显现，并在后续所有天数中持续保留。

## 目标

- 在第三阶段星球顶部模型聚集区域的中间生成一条短小、轻微弯曲、连续一体的低模土路。
- 小路按总天数 `15-17` 逐步形成，在 `17` 天达到完整状态。
- 从 `18` 天开始直到后续所有阶段，持续保留这条小路。
- 不新增外部 `glTF/glb` 资源，直接通过程序生成低模几何体，控制资源体积与运行时开销。

## 非目标

- 不制作分段踏步、石板路或木板路。
- 不把小路延伸成连接多个中心模型的长路径。
- 不引入额外贴图、法线或复杂材质效果。
- 不在第四到第六阶段重新替换成另一条路，或对已生成的小路做反向弱化。

## 视觉与摆放设计

### 场景位置

- 小路位于第三阶段星球顶部模型聚集处的中心。
- 小路不与篝火形成明确连接，只表现“模型布局中间自然形成的一小段土带”。
- 小路长度保持克制，避免变成场景中心视觉元素。

### 形态特征

- 路面为连续一条土带，不拆分为多个踏步块。
- 整体只做轻微弯曲，避免短路径上出现过强的 `S` 型夸张感。
- 边缘允许有轻微不规则起伏，保留低模与自然踩踏的感觉。
- 路两侧允许保留少量草地侵入，避免边界过于“切割化”。

### 视觉风格

- 使用低模土色表现，整体偏暖棕土黄。
- 使用 `flatShading` 保留折面感。
- 不使用贴图，依赖简单材质和几何轮廓表达土路质感。
- 视觉优先级低于第三阶段中心模型布局，不抢戏。

## 时序与累积规则

### 天数映射

- 总天数 `15-17` 位于第三阶段窗口内，对应当前项目中的第三阶段中段。
- 小路只在总天数阈值达到后开始出现，不绑定“阶段内局部天数”。

### 显现过程

- 第 `14` 天及以前：小路不可见。
- 第 `15` 天：出现较短、较窄、存在感较弱的小路雏形。
- 第 `16` 天：在保持位置不变的前提下，增加长度、宽度与存在感。
- 第 `17` 天：达到完整形态。
- 第 `18` 天及以后：维持完整形态，不再回退。

### 累积规则

- 小路一旦形成，进入第四、第五、第六阶段后继续保留。
- 后续阶段默认继承第三阶段已完成的小路表现。
- 除非未来有明确的新需求，不在后续阶段删除、替换或重置该模型。

## 技术设计

### 放置层级

- 将小路放入 `TerrainLayer` 统一管理。
- 原因是小路属于地表构成的一部分，与草地覆盖、石头摆放和地表贴地关系更一致。
- 将其留在地形层中，也可以避免结构层承担过多非建筑职责。

### 模型生成方式

- 使用程序生成的低模网格，不新增 `public/models` 下的资产文件。
- 在局部切平面上定义一条较短的中心线，并基于中心线展开为一条带状网格。
- 控制中心线采样段数在低面数范围内，使轮廓有折线感但不过分粗糙。

### 曲线与尺寸参数

- 采用一条轻微弯曲的短路径，弯曲幅度小于当前预览中的夸张方案。
- 默认宽度为“中间点缀型短路径”的中低存在感宽度，不做宽阔景观道路。
- 实现上预留少量常量参数，例如长度、宽度、边缘抖动幅度、贴地抬升量，便于后续微调。

### 贴地策略

- 小路顶点按星球表面法线贴合到球面上。
- 整体相对地表轻微抬高，避免与草地或地表共面造成闪烁。
- 保持足够低的抬升量，避免产生明显悬空感。

### 材质策略

- 使用单一土色系材质并启用 `flatShading`。
- 允许通过少量顶点扰动或明暗变化增强低模质感。
- 不依赖额外纹理资源。

### 时序驱动方式

- 在 `TerrainLayer.update(input)` 中基于全局 `dayCount` 控制小路显现状态。
- 第 `15-17` 天通过可见性、长度系数、宽度系数或整体缩放参数进行渐进增强。
- `17` 天后直接进入稳定完整状态。

### 生命周期与性能

- 仅新增一个轻量级 `Mesh`。
- 不增加额外模型下载、解析和贴图加载。
- 不添加持续播放的动画逻辑。
- 更新阶段仅根据天数切换可见状态和少量形态参数，运行时开销应接近可忽略水平。

## 测试与验收

### 自动化测试

- 为小路的时序规则补充聚焦测试。
- 至少覆盖总天数 `14`、`15`、`16`、`17`、`18`。
- 验证 `14` 天前不可见、`15-17` 天逐步增强、`18` 天后稳定保留。
- 验证跨阶段后小路仍保留，不会因阶段切换被清空或隐藏。

### 手动验收

- 在本地查看总天数 `15`、`16`、`17`、`22`、`46` 的场景表现。
- 确认小路位于中心，而非错误偏移到篝火或其他结构附近。
- 确认小路较短、轻微弯曲、连续成带，整体不抢第三阶段中心模型布局。
- 确认小路无明显悬空、穿插、闪烁，且不会把周围草地和石头完全压没。

### 回归关注点

- 第二阶段篝火场景不受影响。
- 第三阶段中心模型布局仍为核心主体，小路仅作为辅助细节。
- 第四到第六阶段的建筑、角色与特效展示不受破坏。

## 实施边界

- 本次规格只覆盖第三阶段短土路的设计与接入，不扩展到更多道路系统。
- 如果后续需要“多段路径”“石板路”“通往篝火的主路”等能力，应另开设计，不在本规格中顺带实现。
